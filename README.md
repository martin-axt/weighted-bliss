# Weighted Bliss

A minimalistic todo app that prioritizes tasks using a weighted system (Urgency, Mood, Time).

## Local Development

```bash
# Install dependencies
npm install

# Run in dev mode (Vite)
npm run dev

# Run the server (Express + SQLite)
npm start
```

## Deployment on Raspberry Pi (Docker)

```bash
# Build and run with Docker Compose
docker compose up -d --build
```

Access at `http://<your-pi-ip>:3001`
